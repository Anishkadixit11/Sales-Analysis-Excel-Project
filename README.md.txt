# Sales Analysis & Dashboard Using Excel

## Project Overview
This project analyzes sales, profit, and discount trends across regions, product categories, and shipping methods using Excel.  
The goal is to generate business insights and create a professional dashboard.

## Dataset
The dataset contains order-level data including:
- Order ID – Unique identifier for each order
- Region – Geographic region of the sale (East, West, Central, South)
- Category / Sub-Category – Product classification
- Sales – Total sales value
- Profit – Profit generated per order
- Discount – Discount applied
- Ship Mode – Mode of delivery (Regular Air, Express Air, Delivery Truck)

## Steps Performed
1. **Data Cleaning**
   - Checked for missing values and duplicates
   - Verified consistency of categorical columns
2. **Data Analysis**
   - Created pivot tables for sales, profit, and discounts
   - Summarized trends by Region, Category, Sub-Category, and Ship Mode
3. **Visualization**
   - Created column charts for each analysis
   - Compiled charts into a single dashboard
4. **Insights**
   - Provided business insights based on the charts

## Key Insights
- Central and West regions have the highest sales; East and South are lower.
- Technology and Furniture Supplies lead in sales.
- Binders and Binders Accessories, Office Machines, and Telephones and Communication are top profit-generating sub-categories.
- Regular Air Ship Mode has the highest total discount.

## Outcome
- A clean, interactive Excel dashboard providing quick insights into sales and profit trends.
- Data-driven guidance for business strategy.
